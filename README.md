# Złoty Jeleń – strona WWW

Nowoczesna, responsywna strona domu gościnnego i restauracji **Złoty Jeleń** w Złotym Potoku (Jura Krakowsko-Częstochowska).

## Technologie

- React 19 + TypeScript
- Vite 6
- React Router 7

## Uruchomienie

```bash
npm install
npm run dev
```

Produkcja:

```bash
npm run build
npm run preview
```

## Struktura podstron

| Ścieżka | Opis |
|---------|------|
| `/` | Strona główna |
| `/oferta/pokoje` | Pokoje gościnne |
| `/oferta/szkoly` | Wycieczki szkolne |
| `/oferta/wesela` | Oferta weselna |
| `/oferta/grupy` | Grupy zorganizowane |
| `/menu` | Menu restauracji |
| `/atrakcje` | Atrakcje okolicy |
| `/wypozyczalnia` | Wypożyczalnia |
| `/kontakt` | Kontakt i formularz |

## Zasoby

Logo, zdjęcia hero i grafika jelenia geo znajdują się w `public/images/`.
